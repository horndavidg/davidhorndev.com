Projects:


  Local Me

Local Me is an event scheduling application that can be tailored to a single city for specificity. I developed this app with a classmate and we used Ruby on Rails with Google OAuth, added functionality for posting to Google Calendar as well as picture uploading, and a combination of jQuery with AJAX for single page functionality.

  Rapture Capture

Rapture Capture is a travel journaling application that I developed using Express, Node.js, and MongoDB. People can create accounts and start to save their memories from favorite trips by adding the locations to a personalized map (using Google Map API), uploading photos from the trip (using Uploadcare) and saving specific journal entries to specific days on a given trip. Also allows users to see current weather information and other location specific information. Finally incorporated jQuery and AJAX for single page functionality.
